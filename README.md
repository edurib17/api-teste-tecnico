# API NodeJs Mysql
Teste Técnico para vaga de estágio back-end, API faz o CRUD de clientes e lavouras regristradas.

## Development
Clone the repository:
`git clone git@github.com:edurib17/api_teste_tecnico.git`
<br/>
Insomnia:
`https://insomnia.rest/download/`


### Run
```
npm start
```
